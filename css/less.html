# 变量

@basecolor: #ff6700;
@fsize: 32px;
//在sass中是用$来定义变量

.less{
    color: @basecolor;
    font-size: @fsize + 10px;
}


.test(){  //如果类名后面加个(),它就是单纯用来混合的，最后不回编译出来
    color:red;
}

/*
变量可以 +、-、*、/，运算符的左右两侧必须加个空格
两个数参与运算，
如果只有一个数有单位，最后的结果就以那个单位为准
如果都有单位，结果以第一个单位为准

# 模式匹配

.triangle(@direction, @size, @color) {
  border-style: solid;
  border-color: transparent;
  border-width: @size;
  @base: @size * 2;
  & when (@direction = top) {
    border-bottom-color: @color;
    border-bottom-width: @base;
  }
  // 更多方向的模式匹配...
}

.arrow {
  .triangle(top, 5px, #f00);
}

//======

# 映射

从3.5开始，可以将混合（mixins）和规则集（rulesets）作为一组值的映射（map）使用。

#colors() {
  primary: blue;
  secondary: green;
}

.button {
  color: #colors[primary];
  border: 1px solid #colors[secondary];
}

  color: blue;

    color: green;
  }
  
  &::before{  //伪元素
        
  }
    
  >a{  //子代选择器
    color:black;
  }

# 转义

它允许使用任意字符串作为属性或变量值，内部的任何内容按**~"anything"**原样使用，除了插值没有任何变化。从less3.5+开始就不需要~+引号转义了

@min768: ~"(min-width: 768px)";
.element {
  @media @min768 {
    font-size: 1.2rem;
  }
}

//编译为
@media (min-width: 768px) {
  .element {
    font-size: 1.2rem;
  }
}

//从less3.5开始可简写
@min768: (min-width: 768px);
.element {
  @media @min768 {
    font-size: 1.2rem;
  }
}

    color:lighten(@color,20%);  //lighten函数，颜色变浅20%
